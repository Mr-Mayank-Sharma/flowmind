8.2 — Glossary
Term
Definition
DAG
Directed Acyclic Graph — data structure for representing workflow pipelines without cycles
MCP
Model Context Protocol — standard for tool use between AI models and external services
RAG
Retrieval-Augmented Generation — enhancing LLM responses with retrieved context from memory
BullMQ
Redis-backed job queue for Node.js with priority, retry, delay, and rate limiting built-in
Skill
A reusable, versioned piece of agent capability that the AI learns and improves over time
Context Engine
The RAG + user modeling system that injects relevant memories into every LLM prompt
OS Control Center
The Home section that manages local AI frameworks like Ollama, ComfyUI, LM Studio
Framework
An AI tool installed on the user's OS (e.g., Ollama, ComfyUI, Hermes Agent, OpenCode)
System Bridge
The server-side service with OS-level access that controls local processes and reads metrics
Marketplace Flow
A workflow published by a user (or FlowMind default) available for all users to clone/run
Clone
Copy a marketplace flow into your own workspace — fully editable, never affects the original
tRPC
TypeScript Remote Procedure Call — fully typed API client/server with no code generation needed
Qdrant
Open-source vector database used for semantic search over memories and context
SSE
Server-Sent Events — HTTP-based protocol for streaming tokens from LLM to browser in real time
RBAC
Role-Based Access Control — permissions system: Owner, Admin, Member, Viewer, API Token
JIT Provisioning
Automatic account creation on first SSO login without manual admin setup
